# CS422-MaxGPA-project-grade-distribution-analysis-tool
Project1-CS422

## Team Roles

**Data / ETL**
- Reads the CSV data  
- Cleans and organizes the data  
- Filters data by year  

**Catalog / Degree Requirements**
- Finds required classes for each major  
- Stores those classes in the program  
- Makes sure they match the UO catalog  

**Analytics / Report Generation**
- Calculates grade distributions  
- Compares instructors  
- Generates the final report  

**UI / Integration / Setup**
- Connects all parts of the system  
- Handles user input/output  
- Writes setup instructions (README)
